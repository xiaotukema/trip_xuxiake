合规数据接入

用授权数据增强大模型，而不是违规绕过平台爬取

授权/官方 API 优先

携程、飞猪、同程、航司、酒店、景区、地图等价格和库存应通过开放平台、商务合作或官方 API 接入，再交给大模型做结构化规划。

UGC 来源可追溯

小红书、抖音等社媒内容只支持官方授权、达人合作或用户主动导入；每条拍照点保留来源、更新时间、热度和置信度。

预算可解释

每个方案预算由交通、住宿、餐饮、门票/体验分项累加，Tab 上直接展示总价区间，卡片内展示明细和价格来源状态。

攻略内容不做未授权爬取

授权/官方 API 优先

小红书、抖音、携程等攻略内容应通过开放平台、商务授权、达人合作或用户主动粘贴/导入获得，避免绕过登录、验证码和反爬策略。

可信度标注

每条拍照点、路线建议和预订链接都保留来源、更新时间、置信度与人工审核状态，降低 AI 幻觉和过期信息风险。

隐私保护

行程偏好、预算和定位默认本地保存；同步时应加密传输、最小化采集，并允许用户删除导入内容与位置历史。
